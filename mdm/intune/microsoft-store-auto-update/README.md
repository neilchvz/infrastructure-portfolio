# Windows - Allow Auto-update of Microsoft Store Apps

## Overview
An Intune configuration policy that enables automatic updates for 
Microsoft Store applications on managed Windows devices.

## What this policy configures

- **Auto-update Microsoft Store Apps** — enabled, allowing Store apps 
  to update automatically without user intervention

## Why this policy exists
Ensures Microsoft Store applications stay current with security patches 
and feature updates without requiring manual user action or IT intervention. 
Reduces the attack surface from outdated Store app versions.

## Platform
Built for and deployed via **Microsoft Intune**. Settings catalog policy 
targeting Windows 10 and later.
