# PowerShell Infrastructure Library

Production-grade PowerShell scripts for Microsoft 365, Entra ID, Azure, and Windows endpoint management. Built for Platform and Staff Infrastructure roles at SaaS companies — emphasizing structured output, pipeline compatibility, and NIST-aligned security posture.

24 scripts across 7 categories.

## Categories

### Identity Lifecycle Automation
`powershell/identity-lifecycle/`
End-to-end M365 user lifecycle management — provisioning, offboarding, license reconciliation, and orphaned account detection.

### Zero Trust / Conditional Access
`powershell/zero-trust-ca/`
Conditional Access policy export, drift detection, and gap reporting for NIST and SOC 2 compliance evidence.

### Messaging Infrastructure Ops
`powershell/messaging-ops/`
Exchange Online mailbox delegation auditing, shared mailbox provisioning, and transport rule reporting.

### Data Governance & Compliance Automation
`powershell/data-governance/`
Microsoft Purview retention policy deployment, sensitivity label coverage reporting, and full-volume audit log extraction.

### Hybrid Identity & Directory Ops
`powershell/hybrid-identity-ad/`
Active Directory stale object reporting, group membership reconciliation, and replication health monitoring.

### Endpoint Configuration & Drift Remediation
`powershell/endpoint-configuration/`
Windows endpoint baseline auditing, NIST security baseline application, software inventory collection, and automated drift remediation.

### Azure Infrastructure Automation
`powershell/azure-infrastructure/`
ARM/Bicep template deployment with tagging enforcement, cost anomaly detection, Azure Policy compliance scanning, and NSG security baseline auditing.
